
 FitnessX 

An iOS fitness tracking prototype built with **SwiftUI** and **MapKit**.

 Current Features (Phase 1)
- SwiftUI navigation flow
- Live GPS tracking (CoreLocation)
- Real-time route rendering (MapKit polylines)
- Modular SwiftUI components
- MVVM architecture structure

 Planned Features (Phase 2)
- User account creation + authentication
- Workout history + persistence
- Distance / calorie calculations + analytics dashboard
- HealthKit integration
- Cloud backup + syncing
- Testing & evaluation

 Tech Stack
Swift • SwiftUI • MapKit • CoreLocation • MVVM

 Status
Interim prototype (Phase 1 complete / Phase 2 planned)
